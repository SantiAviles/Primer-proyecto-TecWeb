Bienvenido al sitio web de demostración.

Aquí utilizamos elementos genéricos como 'div' para organizar el contenido de la página de manera flexible.
En la sección 'nav' se utiliza para definir una sección de navegación en una página web, como un menú con enlaces que permiten moverse entre diferentes páginas o secciones del sitio.
Se refiere a la barra lateral izquierda. Suele contener elementos como menús de navegación, enlaces rápidos o widgets adicionales que ayudan a guiar al usuario.
Este div contiene el contenido principal de la página. Suele ser el área más importante donde se muestra la información clave que el usuario necesita ver.
Dentro de "main", este div agrupa un artículo o sección de contenido específico. Se utiliza para organizar bloques de información dentro del contenido principal.
Es una subdivisión dentro del artículo. Aquí puedes colocar una primera parte de contenido relevante, como un párrafo introductorio o una imagen destacada.
Similar a "section-1", pero destinado a contener otra parte del contenido del artículo, como una continuación, detalles adicionales o conclusiones.
Es la barra lateral derecha. Aquí puedes colocar información complementaria como anuncios, noticias, enlaces relacionados o contenido adicional que no es parte del contenido principal.
Representa el pie de página de la web. Es el área en la parte inferior de la página donde se suele incluir información adicional como derechos de autor, políticas, enlaces a redes sociales o detalles de contacto.
